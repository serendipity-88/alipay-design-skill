# 设计意图参考

## 1. 情绪-视觉映射

根据 Pass 0 确定的用户情绪状态，选择对应的视觉策略：

| 场景情绪 | 视觉策略 | 关键手法 | 禁止 |
|----------|----------|----------|------|
| 成功/完成 | 大留白 + 品牌蓝点缀 + 插图 | 40%+ 屏高给情绪头部 | 不堆信息、不加多余 CTA |
| 受阻/需引导 | 暖色激励 + 紧凑信息 + 单 CTA | 一句话说清原因+方案 | 不制造焦虑、不用红色警告 |
| 错误/失败 | 红色提示但克制 + 明确下一步 | 错误文案 ≤ 15 字 | 不用感叹号轰炸、不全屏红 |
| 营销/利益 | 渐变暖色 + 数字突出 + 信任递进 | 先利益再教育 | 不全页促销色 |
| 日常/中性 | 白卡岛屿 + 灰底呼吸 | 卡片间距 ≥ 12px | 不加不必要的色彩装饰 |

## 2. 信息密度路由

根据页面类型选择密度级别，决定卡片数量、信息层级和留白比例：

| 密度级别 | 典型页面 | 组件策略 | 单屏最大信息点 |
|----------|----------|----------|---------------|
| 轻量 | 结果页、确认页、闪屏 | 大插图 + 1 标题 + 1 按钮 | 3 |
| 标准 | 表单、列表、详情、教育引导 | 白卡分区 + 标题行 + 内容 + 操作栏 | 5-7 |
| 密集 | 仪表盘、数据表、管理后台 | ProCard 网格 + ProTable + 筛选 + 统计 | 10+ |

## 3. 品牌色用量约束

- 品牌蓝 `#1677ff` **只用于**：CTA 按钮、关键数字、选中态、链接文字
- 品牌蓝面积**不超过页面 15%**
- **禁止竞品品牌色做强调**：微信绿 `#07C160`、美团黄 `#FFD100`、抖音黑红等
- 成功色 `#00b578` **只用于**成功语义（对勾、完成状态），不做装饰或强调
- 语义色（成功/警告/危险）只在对应语义场景使用，不做配色方案
- 当需要"第二色"区分层次时，用**同色系深浅**（如 `#F5F8FF` vs `#FFFFFF`）或**阴影层级**，不引入新色相
